Programação para Internet I

Prof. Jesiel Viana
@jesielviana

Objetivo Geral

Capacitar o aluno para compreender o funcionamento básico da Web e desenvolver habilidades para projetar e implementar aplicações Web com páginas estáticas seguindo os princípios de usabilidade e boas práticas do desenvolvimento Web

Conteúdo Programático

  1. Introdução aos conceitos de Internet;
  2. Linguagens de Marcação - HTML;
  3. Fundamentos de folhas de estilo - CSS;
  4. Princípios de Design responsivo;
  5. Usabilidade na Web;
  6. Introdução à Javascrip; e
  7. Conceitos Básicos sobre Aplicações Cliente/Servidor.

Metodologia

Problem Based Learning - PBL

  • Aulas expositivas dialogadas (poucas, só algumas conceituais);
  • Aulas práticas (Laboratório);
  • Live Code;
  • Resolução de problemas reais; e
  • Desenvolvimentos de Projetos.
